Class Project: C++ Game

Environment: Visual Studio, C++, SDL2

About

This game was a solo final project for my CS 144 course. The game was a bullet hell where the white square (player) has to avoid the black squares. The game had several phases all of which were in sync with the soundtrack. Every component was custom-made, except the rendering functionality of SDL2. The project requirements were open-ended and to create a project that would challenge yourself. My first introduction to SDL2 was through this project.

Class Project: Java Game

Environment: Eclipse, Java, JavaFX
Github: https://github.com/CS145-Computer-Science-2/finalprogram-buelowheuer

About

This game was a partner final project for my CS 145 course. The project requirements were to create any project that uses JavaFX. My partner and I exceeded the requirements and created a game using the canvas in JavaFX. The game was fully functional, had visual and audio elements, and additionally had a level editor. Besides the heart icon, player sprite, and slime sprite, all other visual and audio elements were made by my partner and I.
